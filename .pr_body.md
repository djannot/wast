## Summary

Implements comprehensive XXE (XML External Entity) vulnerability scanning functionality for WAST. This PR adds a new scanner that detects XXE vulnerabilities through multiple detection methods including in-band file disclosure, error-based detection, and out-of-band (blind) XXE attacks.

Closes #248

## Features Implemented

### 1. **XXE Scanner** (`pkg/scanner/xxe.go`)
- Comprehensive XXE detection with 8 payload types
- In-band detection using file disclosure signatures (`/etc/passwd`, `boot.ini`, `win.ini`)
- Error-based detection using non-existent file access
- Out-of-band/blind XXE detection with callback server support
- Multiple content types tested: `application/xml`, `text/xml`, `application/soap+xml`
- SOAP-specific payloads for web service testing
- Safe mode support (prevents active exploitation during passive scans)
- Evidence extraction from responses for analyst verification
- Finding verification with configurable retries and delays

### 2. **Endpoint Discovery**
- Automatically discovers XML endpoints by testing common paths:
  - `/api/xml`, `/xml`, `/soap`, `/services`, `/ws`, `/api/soap`
- Tests both POST and GET methods
- Respects target URL structure

### 3. **Integration with Existing Infrastructure**
- Seamlessly integrated into parallel scan execution (`pkg/scanner/executor.go`)
- XXE results included in unified scan results (`pkg/scanner/aggregator.go`)
- Risk scoring updated to account for XXE findings
- Priority action recommendations for high-severity findings
- SARIF 2.1.0 format support with CWE-611 mapping (`pkg/output/sarif.go`)

## XXE Detection Payloads

The scanner includes 8 payload types covering major XXE attack vectors:

1. **Linux File Disclosure** - Attempts to read `/etc/passwd`
2. **Windows File Disclosure (boot.ini)** - Attempts to read `C:\boot.ini`
3. **Windows File Disclosure (win.ini)** - Attempts to read `C:\windows\win.ini`
4. **Error-Based Detection** - Triggers XML parsing errors
5. **Blind XXE (Out-of-Band)** - Uses callback server for detection
6. **SOAP Envelope XXE** - Targets SOAP web services
7. **Parameter Entity XXE** - Tests parameter entity expansion
8. **External DTD XXE** - Tests external DTD loading

## Detection Signatures

- `root:x:0:0:` - Linux passwd file signature
- `[boot loader]` - Windows boot.ini signature
- `[extensions]` - Windows win.ini signature
- XML parsing error messages
- Callback server hits for blind XXE
